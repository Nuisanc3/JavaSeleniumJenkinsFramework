# Interview questions on Selenium Java

== Q1 ==
Q1 ->  Why should we prefer selenium webdriver ?
* Open source (Free to use)
* Multi-browser and Multi-platform support
* Selenium can be coded in Java, Python, C# etc. 
* Exclusively for web based apps 
* Selenium is the parent, webdriver is the tool. They release IDE as well. 

== Q2 ==
Q2 -> Which protocol is being used in selenium ? 
* W3C protocol
* The protocol supported by all latest browsers
